《金刚狼3》自断一臂，最完整的删减片段盘点

《金刚狼3》国内上映版删减14分钟之久，大家似乎已经“坦然接受”了。从另一个角度来看，R级片进入国内院线并且和北美同时上映已经是相当不容易的事了，所以好像删了14分钟也是理所应当的事。

但是只谈删减，不谈内容都是耍流氓。从《金刚狼》，到《全冈良》，我们必须先要知道这14分钟究竟是哪些内容，是不是真的不适合国内院线上映播放，然后才能给出结论删减得是否合理。

Quick Start

当然这个彩蛋应该只是北美特供版，并不只有我大中华地区没有，在全球范围内的其他国家也都没有，包括当时在柏林电影节首映的时候。所以现行的《金刚狼3》大概有三个版本，137分钟的北美版，135分钟的柏林版和全球版，以及123分钟的中国大陆版。

至于彩蛋的内容这几天也在网上广为流传了。总结来说就是死侍小贱贱路见不平，准备拔刀相助，但是超级英雄都喜欢变个装再出手，最后耽误了时间，导致路人被枪杀。小贱贱走过去躺在死者的尸体上，开启了嘴炮模式。

其中有几个不得不提的亮点。

镜头中的电影院挂着LOGAN正在上映的海报，而且小贱贱中途打电话的那位赖瑞正是他电影中的服装道具师；

史丹·李再一次按照惯例出来露了个脸，并且有一句非常搞笑的台词：Nice Suit.

小贱贱躺在尸体上还吐槽说到：如果你是金刚狼就不会死了。不过，看过《金刚狼3》 的朋友可能就不会这么以为了。

以及最后的音乐是超人中的经典配乐。很明显是在黑：为什么超人每次换衣服都这么快。
